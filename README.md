# JviewUI

## First
```
npm install nhw-public-component-ui --save
```

## Second -  main.js 
```
import NHWUI from 'nhw-public-component-ui'
import 'nhw-public-component-ui/dist/nhw-public-component-ui.css'
Vue.use(NHWUI)
```
## 组件使用，例如：
```
<template>
  <JSplit>
    <div slot="left">我是左边</div>
    <div slot="right">我是右边</div>
  </JSplit>
</template>
```
